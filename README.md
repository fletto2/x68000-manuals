# X68000 Manuals — English translations

English translations of the following Sharp X68000 technical books, translated from the
Japanese originals and cross-checked against the relevant component datasheets:

- **Inside X68000** (1992)
- **Outside X68000** (1993)
- **X68000 Technical Guide**
- **X68000 Service Manual** (CZ-634C / CZ-644C)
- **Sharp Hard Disk Unit IT-X640 / IT-X680 User's Manual**

Each manual is provided as **PDF** and **ODT**, with the source **Markdown** under
[`markdown/`](markdown). The original schematics, timing charts, block diagrams and other
figures — which do not survive as text — are reproduced from the source pages alongside
their translated captions. The [`datasheets/`](datasheets) folder holds the datasheets for
the X68000's peripheral ICs that were used to verify the hardware register, address and
bit-field details — MC68901 (MFP), HD63450 / MC68450 (DMAC), MB89352 (SPC / SCSI),
RP5C15 (RTC), YM2151 (OPM), µPD72065 / µPD765 (FDC), MSM6258 (ADPCM) and YM3802 (MCS / MIDI).

These are datasheet-backed translations, provided for preservation and reference. All
trademarks belong to their respective owners.
